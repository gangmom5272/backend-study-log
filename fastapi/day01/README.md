# FastAPI Day 01

## 학습 범위
- 쿼리 매개변수와 문자열 검증
- 경로 매개변수와 숫자 검증

## 핵심 개념

### 쿼리 매개변수와 문자열 검증
- 쿼리 매개변수는 URL에서 `?key=value` 형태로 전달되는 값이다.
- FastAPI에서는 `Query()`를 사용해 쿼리 매개변수의 조건을 설정할 수 있다.
- 문자열 길이, 정규식 패턴, 기본값, 필수 여부 등을 검증할 수 있다.

### 경로 매개변수와 숫자 검증
- 경로 매개변수는 URL 경로 안에 포함되는 값이다.
- FastAPI에서는 `Path()`를 사용해 경로 매개변수의 조건을 설정할 수 있다.
- 숫자 값에 대해 `gt`, `ge`, `lt`, `le` 같은 조건을 줄 수 있다.

## 실습 내용
- `Query()`를 사용해 문자열 길이 검증하기
- 필수 쿼리 매개변수와 선택 쿼리 매개변수 구분하기
- `Path()`를 사용해 숫자 범위 검증하기
- 경로 매개변수와 쿼리 매개변수를 함께 사용하는 API 작성하기
